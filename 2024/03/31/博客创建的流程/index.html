博客创建的流程

2024/4/1 笔记 hexo next vercel cloudflare cdn

记录一下 $blog 创建、换主题、部署和加速的大体流程。

最开始是跟着枫叶的教程，用 Hexo + Next 搭出了第一版博客。

  1. 从零开始搭建个人博客（超详细） - 知乎

教程整体比较完整。如果某一步报错，可以先看评论区，很多细节问题已经有人补充过。

  1. 在 Themes | Hexo 里选择自己喜欢的主题，下载后放到 $blog/themes 目录，再在配置里切换主题。

第一个入口是 GitHub 中的下载地址，点进去点击 Download 即可：

第二个入口是主题的展示效果：

  1. 常用工具记录。

在线调色板：用于获取颜色值，方便修改主题配色。

高清 4K 壁纸：可以作为博客背景图素材来源。

FontAwesome 图标库：用于博客图标配置。

Cloudflare 免费版教程：用于配置 CDN 加速。Cloudflare 里还有不少速度优化选项，可以后续按需打开。

Hexo 博客站点加速不完全指南：介绍了 Vercel 等补充加速方案。

PageSpeed Insights：用于测试页面加载速度，并查看具体优化建议。
